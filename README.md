# EnbyLoo

A resource for non-binary folks to find a safe place to rest. 

## Features
- REFUGE API database
- Google Maps component 
- Search feature
- Add, edit, delete, show, index restroom locations
- About page 
- Resource list

## Build
- Ruby on Rails backend
- Postgres DB
- Angular.JS front end
- SASS styling
